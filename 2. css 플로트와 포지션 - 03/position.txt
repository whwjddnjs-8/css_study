1. 부모요소 relative, 자식요소 absolute
2. 포지션 속성의 기본값(default)는 position: static
3. 포지션 속성이 없으면 left, right, top, bottom, z-index 를 사용할 수 없음. 곧, 포지션 속성과 반드시 같이 사용(static 제외)

- 포지션 속성 : static, relative, absolute, fixed, sticky
static : position 속성을 없음
absolute : 요소가 절대적인 위치에 배치
relative : 요소가 상대적인 위치에 배치
fixed : 요소가 브라우저 창에 고정되어 배치
